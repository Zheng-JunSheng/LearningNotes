# 一些快捷键

## 番茄助手

| Shift + Alt + O     | 查找文件                                  |
| ------------------- | ----------------------------------------- |
| **Shift + Alt + S** | **查找某个变量名**                        |
| **Shift + Alt + F** | **找到所有引用某个字符串的地方**          |
| **Alt + G**         | **在声明和定义之间互相跳转**              |
| **Alt + O**         | **在`.h`文件和`.cpp`文件之间互跳**        |
| **Shift + Alt + R** | **变量重命名,会找到所有使用该变量的地方** |
